Instalar como app

Instala esta app en tu teléfono o PC para acceder sin navegador, como una app nativa.

En Chrome/Edge: busca el ícono de instalación en la barra de direcciones. En iOS Safari: comparte → "Añadir a inicio".

Guía: configurar Railway

Sigue estos pasos para activar la sincronización automática con CMI Escolar:

  1. Crea cuenta en railway.app con tu cuenta de GitHub
  2. Crea un New Project → Deploy from GitHub repo
  3. Sube la carpeta backend/ a un repositorio de GitHub
  4. En Railway, ve a Variables y agrega:
      CMI_RUT = tu RUT sin guión (ej: 22837339k)
      CMI_PASS = tu contraseña de CMI Escolar
  5. Copia la URL que Railway te da (ej: https://xxx.up.railway.app)
  6. Pégala en el campo de arriba y guarda

El plan gratuito de Railway incluye $5 USD de crédito/mes. Este backend consume menos de $0.50/mes.
